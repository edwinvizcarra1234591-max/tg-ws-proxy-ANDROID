## 📋 About tg-ws-proxy-ANDROID

This application helps users connect to Telegram by creating a proxy server. It works on Android devices but can also support other platforms with similar setup. The proxy uses the SOCKS5 protocol to route your Telegram data safely through a web socket server.

The tool focuses on ease of use while applying Flowseal proxy logic for stable and secure communication. It is useful if your local network restricts direct access to Telegram or if you want to add an extra layer of privacy.
